# Bootcamp Santander desafio 2

Criaçao de script de Ransonware e de Keylogger


Neste desafio foi criado um script para criptografar os arquivos dentro de uma pasta, e depois solcitando a vitima o pagamento para restaurar os arquivos encriptados e criar o script para descriptografar. Simulando um ataque de Ransonware.

Foi solicitado tambdem a criação de um script de Keylogger, utilizado para capturar tudo que é digitado pelo usuario e armazenar em um arquivo.
 - Para o keylogger é necessario instalar a biblioteca *pynput* do Python. 

-- Estes scrips são para fins academicos e nao devem ser utilizados para tentativa de roubo de informações e devem ser utilizados em ambiente controlado. --
